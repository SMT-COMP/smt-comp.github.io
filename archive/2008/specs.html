Machine Specifications

The competition will run on a cluster of 9 machines at Washington University in St. Louis. For SMT-COMP’07, the specifications of these machines are:

  • 2.4 GHz AMD Opteron 250, configured for single core, 32-bit processing.
  • 1 MB cache.
  • 3.5 GB main memory.
  • Linux, kernel version 2.6.9-55.EL.
  • CentOS release 4.5 (Final)
  • glibc-2.3.4-2.36
  • gcc-3.4.6-8 (if a solver needs a higher version of GCC, this can be accomodated)

However, please note that we may make software and configuration changes before the competition this year. When we release the rules for SMT-COMP’08 we'll decide on the final configuration of the competition cluster.
